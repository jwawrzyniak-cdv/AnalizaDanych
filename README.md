## Jak korzystać z materiałów

1. **Przed zajęciami** — przejrzyj prezentację (`labXX_prezentacja.md`) jako wprowadzenie do tematu.
2. **Na zajęciach** — realizuj zadania z instrukcji (`labXX_instrukcja.md`).
3. **Po zajęciach** — spróbuj rozwiązać zadania rozszerzone (`labXX_rozszerzone.md`).
